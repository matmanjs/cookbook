测试

这里设置爬取的数据等于预设置的数据即可。这里有两种测试方案：有数据时和没有数据时

无数据校验

测试无数据的测试脚本如下：

it('数据快照校验通过', function () {
            expect(data).to.eql({
                'transactionListInfo': {
                    'emptyPic': 'http://now.qq.com/img/nopkdata@2x_c3c9fbba.png',
                    'emptyWording': '暂无流水记录',
                    'isExist': true,
                    'list': [],
                    'total': 0
                }
            });
        });
